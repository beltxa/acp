# Java Spring Overlay Example

This folder contains a minimal Spring-style controller example using ACP framework wrappers:

- `OverlayHttpRuntime` (`org.acp.client.framework`)
- existing ACP well-known model (`/.well-known/acp`)
- outbound ACP-over-HTTP helper with well-known bootstrap

File:

- `OverlayControllerExample.java`

The example is intentionally lightweight and serves as an integration template for existing Spring services.
