
# ACP Architecture & Adoption Overview

This document summarizes how ACP can be deployed and adopted in both **overlay mode**
(low‑friction adoption) and **native mode** (full ACP architecture).

It also shows how ACP separates:

- Agent identity
- Endpoint trust
- Enterprise key custody
- Transport and relay infrastructure

This layered design allows ACP to integrate with existing enterprise infrastructure
without requiring immediate architectural replacement.
