# ACP Java SDK (`io.acp:acp-sdk`)

Status: `Available from repo`

Java SDK implementation for ACP identity, message envelopes, and delivery transports.

## Build and test

```bash
cd sdks/java
mvn test
```

## Maven coordinates (repo-aligned)

```xml
<dependency>
  <groupId>io.acp</groupId>
  <artifactId>acp-sdk</artifactId>
  <version>0.1.0</version>
</dependency>
```
