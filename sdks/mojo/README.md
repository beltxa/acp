# ACP Mojo SDK Wrapper (`acp-sdk-mojo`)

Status: `Available from repo`

Mojo-facing wrapper over the ACP Python runtime.

## Install from repo

```bash
pip install -e sdks/mojo -e sdks/python
```

## Run example

```bash
cd sdks/mojo
mojo examples/overlay_http_client.mojo
```

## First-run reference

For a verified ACP ping flow before Mojo integration, run:

```bash
./getting-started/quickstart_ping.sh
```
