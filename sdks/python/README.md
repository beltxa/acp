# ACP Python SDK (`acp-sdk`)

Reference Python SDK for the Agent Communication Protocol (ACP).

## Install

From source:

```bash
pip install -e .
```

From PyPI (target package name):

```bash
pip install acp-sdk
```

## CLI

The CLI is packaged separately as `acp-cli`.

## Quick Check

```bash
python -c "import acp; print('acp-sdk import ok')"
```
